# BranchScope for Claude

Claude.ai の会話ツリーを可視化・探索する Chrome 拡張機能。

会話中の分岐（メッセージ編集・再生成）をツリー構造で表示し、任意のブランチにジャンプしてそこから会話を続けることができます。

## 機能

- **ツリー可視化** — 会話の全分岐を D3.js でツリー表示
- **ブランチジャンプ** — ノードをクリックして Claude.ai 上でそのブランチに移動し、続きから会話可能
- **テキスト検索** — 会話内容を全文検索、ヒットしたノードをハイライト
- **ブランチフィルタ** — メッセージ数しきい値で枝を非表示
- **プルーニング** — 不要な枝を除外してルート管理
- **テキストリーダー** — 各セグメントの会話全文を閲覧・保存

## インストール

1. このリポジトリをクローン
2. `chrome://extensions` を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」→ `extension/` フォルダを選択

## 使い方

1. Claude.ai でチャットを開く
2. 拡張機能アイコンをクリック → ツリービューが新しいタブで開く
3. 操作:
   - **ラベル** クリック → テキストリーダーで全文表示
   - **ノード** クリック → 枝を折りたたむ/展開
   - **エッジ** クリック → 枝をプルーニング
   - **↗ Continue here** → Claude.ai でそのブランチにジャンプ

## ファイル構成

```
extension/
├── manifest.json      # 拡張機能マニフェスト
├── background.js      # アイコンクリック → viewer タブを開く
├── content.js         # Claude.ai ページ上で動作（データ取得・ブランチ操作）
├── common.js          # 共通ロジック（ツリー構築・D3描画・検索・ナビゲーション）
├── common.css         # 共通スタイル
├── viewer.html/js     # フルページビューア
├── sidepanel.html/js  # サイドパネル版
└── d3.v7.min.js       # D3.js
```

## 必要な権限

- `activeTab` / `tabs` — Claude.ai タブへのアクセス
- `scripting` — ブランチナビゲーション用スクリプト注入
- `cookies` — Claude.ai API 認証
